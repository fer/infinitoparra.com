# infinitoparra.com

Hexo website

### Install

To clone the project and serve your site locally under [http://localhost:4000/](http://localhost:4000/):

```bash
git clone https://github.com/fer/infinitoparra.com.git
cd infinitoparra.com
npm install
hexo serve
```

### Deploy

```
hexo clean ; hexo deploy
```
